# Damoang Classic Theme

다모앙의 클래식 기본 테마입니다.

## 특징

- 전통적인 커뮤니티 레이아웃
- 깔끔한 디자인
- 높은 가독성
- 반응형 지원

## 설정

### Appearance
- **Primary Color**: 메인 색상 (기본값: #10b981 - 그린)
- **Secondary Color**: 보조 색상 (기본값: #3b82f6 - 블루)
- **Show Breadcrumb**: 브레드크럼 표시 여부

### Layout
- **Sidebar Position**: 사이드바 위치 (left/right)
