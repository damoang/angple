# Modern Dark Theme

모던하고 세련된 다크 테마입니다.

## 특징

-   눈의 피로를 줄여주는 어두운 색상
-   세련된 모던 디자인
-   야간 모드에 최적화

## 설정

### Appearance

-   **Primary Color**: 메인 색상 (기본값: #1e293b - 다크 그레이)
-   **Secondary Color**: 보조 색상 (기본값: #475569 - 슬레이트)
-   **Show Banner**: 배너 표시 여부
