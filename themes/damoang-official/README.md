# Damoang Official 2026 Theme

다모앙 공식 테마 2026 - 모던하고 세련된 커뮤니티 플랫폼 디자인

## 특징

### 🎨 디자인

-   **3단 레이아웃**: 왼쪽 네비게이션, 중앙 콘텐츠, 오른쪽 위젯
-   **브랜드 색상**: 보라색(#8b5cf6)과 핑크(#ec4899) 그라데이션
-   **모던 UI**: 둥근 모서리, 그림자, 글래스모피즘 효과
-   **반응형**: 모바일/태블릿/데스크톱 완벽 대응

### 🚀 주요 기능

-   **Fixed 헤더**: 스크롤해도 항상 상단에 고정
-   **실시간 검색**: 상단 검색바로 빠른 게시글 검색
-   **트렌딩 위젯**: AI가 선정한 HOT 토픽 표시
-   **공지 배너**: 중요 공지사항 상단 노출
-   **실시간 인기글**: 오른쪽 위젯에서 인기글 확인
-   **트렌딩 태그**: 실시간 인기 태그 표시
-   **실시간 접속자**: 현재 접속 중인 사용자 수 표시

### 📱 컴포넌트

-   `official-header.svelte`: 검색, 알림, 프로필이 있는 모던 헤더
-   `official-footer.svelte`: 소셜 링크와 정보가 있는 푸터
-   `right-widget.svelte`: 인기글, 태그, 활동 위젯
-   `trending-widget.svelte`: 트렌딩 토픽 배너 (Slot: content-before)
-   `notice-banner.svelte`: 공지사항 배너 (Slot: header-after)

### ⚙️ 설정 옵션

-   **브랜드 색상**: 메인/서브 색상 커스터마이징
-   **공지 배너 표시**: ON/OFF 설정
-   **다크모드**: 다크모드 토글 활성화
-   **헤더 스타일**: Fixed 또는 Static 선택
-   **오른쪽 위젯**: 표시/숨김 설정
-   **트렌딩 기능**: 트렌딩 게시글 표시 여부
-   **실시간 접속자**: 접속자 수 표시 여부

## 디자인 철학

> "함께 모이는 즐거움"

다모앙 Official 2026 테마는 사용자들이 편안하게 소통할 수 있는 공간을 제공합니다.
보라색과 핑크의 따뜻한 색상, 부드러운 그라데이션, 그리고 직관적인 UI로
누구나 쉽게 사용할 수 있는 현대적인 커뮤니티 경험을 선사합니다.
