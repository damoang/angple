## P0: 즉시 필요 (트래픽 전환 전 필수)

### P0-1. 인증/회원 시스템 완성

소셜로그인은 구현 완료. 나머지 인증 흐름 마무리.

| # | 기능 | 설명 | 난이도 |
|---|------|------|--------|
| 1 | 회원가입 | 소셜 신규 가입자 → 닉네임/약관동의 폼 → g5_member INSERT | 중 |
| 2 | 비밀번호 찾기/재설정 | 이메일 기반 재설정 링크 발급 (직홍 연동 계정용) | 중 |
| 3 | 소셜 계정 연결/해제 | 마이페이지에서 소셜 계정 추가 연결/해제 관리 | 중 |
| 4 | 본인인증 (inicert/SIWA) | 필요 시점에 판단 — 당장은 소셜 인증으로 대체 가능 | 높 |
| 5 | 이메일 인증 | 회원가입 시 이메일 확인 메일 발송 | 중 |

### P0-2. 게시글 핵심 기능

일반 사용자가 매일 쓰는 기능들.

| # | 기능 | 설명 | 난이도 |
|---|------|------|--------|
| 1 | 태그 시스템 UI | 글쓰기/수정 시 태그 입력, 게시판에서 태그 필터링, 태그 클라우드 | 중 |
| 2 | 자동저장 (autosave) | 글쓰기 중 주기적 localStorage/서버 저장, 복구 UI | 중 |
| 3 | 비밀글 처리 | is_secret 플래그 기반 접근 제어 (작성자/관리자만 열람) | 낮 |
| 4 | 링크 필드 (link1, link2) | 글쓰기 폼에 링크 입력란 추가, 뷰에서 표시 | 낮 |
| 5 | 카테고리 변경 | 글 상세에서 카테고리 드롭다운 변경 (관리자) | 낮 |
| 6 | 글 이동 (게시판 간) | 관리자가 글을 다른 게시판으로 이동 | 중 |
| 7 | 일괄 삭제/이동 | 체크박스 선택 → 일괄 작업 | 중 |
